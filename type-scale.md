# Type Scale

- 다양한 크기의 텍스트를 사용할 수 있도록 스타일에 따라 구분하여 정의한다.
- 반응형에 대응하기 위하여 폰트 구현을 개발 시 코드상에서는 rem 값을 사용한다.

---

## Display

> Display는 화면에서 텍스트를 가장 주요하게 나타낼 때 사용한다.

| Style  | Size | Font weight | Line height | Letter spacing    |
|--------|------|-------------|-------------|-------------------|
| XLarge | 80   | 400 - 700   | 120         | -3% / -2.4px      |
| Large  | 72   | 400 - 700   | 100         | -2.8% / -2.018px  |
| Medium | 64   | 400 - 700   | 96          | -2.6% / -1.664px  |
| Small  | 56   | 400 - 700   | 84          | -2.4% / -1.344px  |

---
